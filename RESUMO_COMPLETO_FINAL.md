## 🧠 Machine Learning Models

### 1. Churn Prediction
```
RFM Score = (100-r_score)*0.30 + (100-f_score)*0.25 + (100-m_score)*0.20 
            + (100-fidelity_score)*0.15 + (100-engagement_score)*0.10
```

### 2. Demand Forecasting
```
demand = base_sales × seasonal_factor × trend_factor × anomaly_factor
seasonal_factor = weekend(1.3) or weekday(0.9)
```

### 3. Loss Rate Analysis
```
loss_rate = lost_units / (sold_units + lost_units)
risk_score = loss_rate × product_value × storage_temperature
```

### 4. Brand Affinity
```
affinity = common_purchases / total_purchases
correlation = purchases_together / expected_random
```

### 5. Gondola Optimization
```
position_priority = sales_per_hour × profit_margin × visibility_factor
layout_efficiency = total_sales / shelf_space_used
```

### 6. Purchase Optimization
```
EOQ = sqrt((2 × D × S) / H)
Safety_Stock = Z × σ × sqrt(L)
Reorder_Point = (D × L) + Safety_Stock
```

### 7. Behavioral Variations
```
50 Exatamente 5 categorias:
- Temporal (10): Por hora, dia, semana, sazonalidade, eventos
- Product (10): Por categoria, marca, tipo, preço, promoção
- Customer (10): Novo, regular, VIP, em risco, dormindo
- Fidelity (10): Frequência, recência, valor, lifetime, churn_risk
- Predictive (10): Next_buy, cross_sell, upsell, downsell, retention
```
