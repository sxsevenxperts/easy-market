# 🔓 Como Desbloquear o Secret no GitHub

## Situação Atual
- Página aberta: GitHub Secret Scanning Unblock
- Status: Mostra erro 404

## Solução Alternativa Rápida

### Opção 1: Feche e reabra a aba
1. Feche a aba do GitHub
2. Copie este link:
```
https://github.com/sxsevenxperts/easy-market/settings/security?tab=secret-scanning
```
3. Cole na barra de endereço
4. Procure por "Openweather API Key"
5. Clique em "Allow"

### Opção 2: Vá direto para Security Settings
1. GitHub > easy-market
2. Clique em "Settings"
3. Clique em "Security" (no menu esquerdo)
4. Clique em "Secret scanning"
5. Encontre "Openweather API Key"
6. Clique em "Allow push" ou "Unblock"

### Opção 3: Via Menu Settings
1. Acesse: https://github.com/sxsevenxperts/easy-market
2. Clique em "⚙️ Settings" (no topo)
3. Clique em "🔐 Security and analysis" 
4. Encontre a seção "Secret scanning"
5. Encontre "Openweather API Key"  
6. Clique no botão para desbloquear

## Após Desbloquear
Avise-me e farei:
```
git push origin main
```
